📊 Financial PDF Analyzer

90s-style retro web app for uploading company financial PDFs and receiving AI-generated analysis.

🔍 This is a Static Preview

GitHub Pages cannot run Flask/Python backends. The full application with PDF upload and Gemini AI analysis requires a backend server.

To run the full app:

  1. Go to repository
  2. Click Code → Codespaces → Create codespace on branch
  3. In the terminal, run: 
    export GEMINI_API_KEY="your-key"
    python app.py
  4. Open the forwarded port 8000 in your browser
✨ Features
  • Multi-file PDF upload (up to 40 files, 15 MB each)
  • Server-side PDF validation (magic bytes + size checks)
  • CSRF protection & rate limiting
  • Secure security headers (CSP, X-Frame-Options)
  • AI financial analysis via Google Gemini
  • Temporary file storage (auto-deleted)
